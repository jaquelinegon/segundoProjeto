A página inicial do projeto apresenta um menu de navegação bem destacado, posicionado na parte superior da página. Esse menu oferece links que direcionam os usuários para outras páginas importantes do projeto. Ao clicar em qualquer um dos itens do menu, os usuários serão redirecionados para as respectivas páginas, permitindo uma navegação fácil e intuitiva pelo conteúdo do projeto. Além disso, no botão 'informações', 
os usuários encontrarão detalhes sobre os conceitos de PHP e banco de dados aplicados no projeto, oferecendo uma visão aprofundada sobre sua estrutura e funcionamento.
